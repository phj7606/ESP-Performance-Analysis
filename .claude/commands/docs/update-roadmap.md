ROADMAP.md의 완료 작업 체크박스를 업데이트하세요.

## 실행 절차

### 1단계: 소스 읽기
- `docs/ROADMAP.md` 전체 읽기
- `.claude/shrimp_data/tasks.json` 읽기 (completed 상태 태스크 목록 추출)

### 2단계: 체크박스 업데이트 규칙
ROADMAP.md의 각 태스크 항목(`- [ ] **[태스크ID]...`):
- Shrimp tasks.json에 `"status": "completed"` 태스크가 있고, 해당 태스크 설명이 ROADMAP 항목과 대응되면 `- [x]`로 변경
- 매칭 기준: 태스크 내용의 핵심 키워드(파일명, API 엔드포인트, 컴포넌트명)로 판단
- 이미 `[x]`인 항목은 그대로 유지

### 3단계: Phase 헤더 이모지 업데이트
각 Phase의 모든 태스크가 `[x]`이면 → `✅ Phase N`
일부 완료면 → `🔄 Phase N`
전혀 없으면 → `⬜ Phase N`

### 4단계: 진행 추적 통계 갱신
"전체 태스크 수" 표와 "완료율" 수치를 실제 `[x]` 개수로 업데이트

### 5단계: 파일 저장
Edit 도구로 변경사항만 정밀하게 적용 (불필요한 전체 재작성 금지)

### 6단계: 결과 보고
변경된 항목 목록과 Phase별 완료율을 사용자에게 요약 출력
